# shipmnts-sde-round-1
Public repository for Shipmnts SDE role round 1

For running the Server:

1. Configure .env file with your MONGO_URI and PORT
2. Run npm install
3. Run npm run dev OR npm run start

APIs available

1. /store - POST
2. /store/:store_location - PUT

3. /plan - POST
4. /plan/:plan_id - GET
